# Fluxo Placas de Rede de Distribuição Cemig 

Este projeto consiste em um script desenvolvido em Python para otimizar e automatizar processos internos na rede de distribuição de energia onde atuo pela **CSRT ENERGIA**. 

# Contexto e Problema
No ambiente operacional e de logística da rede de distribuição, o controle, triagem ou validação manual de fluxos (como identificações, placas ou registros operacionais) 
pode gerar gargalos, duplicações de registros e erros por falta de rastreabilidade (como não saber com qual colaborador ou equipe está determinada nota técnica). 

Para resolver esse problema, desenvolvi esta automação com o objetivo de acelerar o processamento dessas informações, eliminar erros manuais e trazer controle total sobre o andamento de cada demanda.

# Funcionalidades

Automação de Fluxo: Processamento e triagem lógica automatizada dos dados de entrada e saída.
Validação Rápida: Redução do tempo gasto na verificação manual de registros e placas.
Rastreabilidade e Status: Controle em tempo real para identificar se uma nota já foi executada ou não, quem será o responsável técnico pela instalação e em qual data ela foi ou será realizada.

# Tecnologias Utilizadas
Python: Linguagem principal utilizada para o desenvolvimento de toda a lógica de validação, regras de negócio e automação do fluxo.
Banco de Dados (MySQL): Utilizado para dados, garantindo o armazenamento seguro, histórico das instalações e evitando a duplicidade de notas.
